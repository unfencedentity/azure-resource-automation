# Azure Resource Automation ⚙️

This project automates Azure resource creation using Azure CLI.

## 🧰 Requirements
- Azure CLI installed  
- Active Azure account  

## 🚀 Usage
1. Login to Azure:
az login

2. Run the script:
./create_vm.ps1
3. The script creates:
- A Resource Group  
- A Virtual Machine  
- Displays the VM's public IP address
